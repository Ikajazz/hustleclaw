# Setup Facebook Page - Shopee Affiliate

## 📋 Checklist Setup

### 1. Buat Facebook Page
**Nama Page (pilih salah satu):**
- "Belanja Hemat Shopee"
- "Shopee Deals Indonesia"
- "Rekomendasi Produk Shopee"
- "Belanja Cerdas ID"

**Kategori:** Shopping & Retail / Product/Service

**Bio/About:**
```
🛍️ Rekomendasi produk terbaik di Shopee
💰 Diskon & promo update setiap hari
⭐ Produk pilihan dengan rating tinggi
📦 Trusted seller only

Follow untuk update produk viral & hemat!

#ShopeeAffiliate #BelanjaHemat #PromoShopee
```

**Profile Picture:** Logo simple (bisa pakai Canva)
**Cover Photo:** Banner promo menarik

### 2. Setup Meta Business Suite
1. Buka business.facebook.com
2. Link Facebook Page
3. Setup Instagram (optional, tapi recommended)
4. Enable insights & analytics

### 3. Page Settings
**General:**
- ✅ Enable messages (auto-reply)
- ✅ Enable reviews
- ✅ Enable recommendations

**Messaging:**
- Setup instant reply:
  ```
  Halo! Terima kasih sudah hubungi kami 😊
  
  Untuk info produk & link pembelian, cek post terbaru kami ya!
  
  Ada pertanyaan? Chat aja langsung!
  ```

**CTA Button:** "Shop Now" atau "Send Message"

### 4. Join Facebook Groups (20-30 grup)
**Target grup:**
- Grup jual-beli lokal (Jakarta, Surabaya, Bandung, dll)
- Grup ibu rumah tangga
- Grup mahasiswa
- Grup komunitas (hijab, makeup, gadget, dll)
- Grup belanja online Indonesia

**Cara cari:**
1. Search: "jual beli [kota]", "belanja online", "shopee indonesia"
2. Filter: Groups, Members 10K+
3. Join & tunggu approval
4. Baca rules grup!

### 5. First Posts (Warm-up)
**Jangan langsung jualan!** Post 3-5 konten value dulu:

**Post 1 (Introduction):**
```
Halo semuanya! 👋

Page ini dibuat untuk sharing rekomendasi produk terbaik di Shopee.

Kami riset produk dengan:
✅ Rating tinggi (4.5+ bintang)
✅ Terjual ribuan
✅ Harga terbaik
✅ Seller terpercaya

Follow untuk update produk viral & promo menarik setiap hari!

#ShopeeAffiliate #BelanjaHemat
```

**Post 2 (Tips):**
```
💡 Tips Belanja Hemat di Shopee:

1. Cek rating & review dulu
2. Bandingkan harga dari beberapa seller
3. Manfaatkan voucher & cashback
4. Belanja saat flash sale (00:00, 12:00, 18:00)
5. Follow page ini untuk rekomendasi produk terbaik!

Ada tips lain? Share di comment! 👇
```

**Post 3 (Engagement):**
```
🤔 Kalian lebih suka belanja produk apa di Shopee?

A. Fashion (baju, sepatu, tas)
B. Beauty (makeup, skincare)
C. Gadget (HP, aksesoris)
D. Home & Living (perabotan rumah)

Comment A/B/C/D ya! Nanti kita share rekomendasi produk sesuai pilihan kalian 😊
```
