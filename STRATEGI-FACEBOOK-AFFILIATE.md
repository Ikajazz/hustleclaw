# Strategi Auto Affiliate Shopee - Facebook

**Target:** Komisi 1 juta+/hari dari Shopee Affiliate via Facebook
**Platform:** Facebook Page, Groups, Marketplace, Stories
**Niche:** Produk trending dengan komisi tinggi (fashion, elektronik, home & living)

---

## 🎯 Fase 1: Setup & Riset (Hari 1-3)

### 1. Setup Akun & Page
- Buat Facebook Page niche-specific (contoh: "Belanja Hemat Shopee", "Gadget Murah Indonesia")
- Optimasi bio dengan CTA jelas
- Setup Meta Business Suite untuk analytics
- Gabung 20-30 grup jual-beli aktif (ibu rumah tangga, mahasiswa, komunitas lokal)

### 2. Riset Produk (Otomatis)
**Tools:** Tavily web search + SEO skill

**Kriteria produk:**
- Komisi ≥6% (produk bertanda "Komisi XTRA")
- Rating ≥4.5 bintang
- Terjual 1000+ bulan ini
- Harga Rp50.000 - Rp500.000 (sweet spot)
- Trending di TikTok/Instagram

**Kategori prioritas:**
1. Fashion wanita (baju, hijab, tas)
2. Gadget & aksesoris HP
3. Skincare & makeup
4. Perlengkapan bayi
5. Home decor

**Script riset otomatis:**
```bash
# Cari produk trending
# Input: kategori
# Output: list produk + link affiliate + komisi
```

## 📈 Fase 3: Distribution & Engagement (Ongoing)

### 1. Facebook Page Strategy
**Content mix:**
- 60% produk affiliate (hard sell)
- 30% tips & edukasi (soft sell)
- 10% engagement (poll, quiz, giveaway)

**Boost engagement:**
- Reply semua comment dalam 5 menit (auto-reply bot)
- Pin best-performing post
- Run mini giveaway (syarat: like, comment, share)

### 2. Facebook Groups Strategy
**Rules:**
- Baca rules grup (jangan langsung spam)
- Kontribusi value dulu (jawab pertanyaan, kasih tips)
- Post produk 1-2x/hari max per grup
- Variasi caption (jangan copy-paste sama)

**Teknik:**
- Soft sell: "Aku baru beli ini, bagus banget! [link]"
- Problem-solution: "Ada yang cari [produk]? Aku nemu yang murah nih [link]"
- Testimonial style: "Review jujur [produk] - worth it gak sih?"

### 3. Facebook Marketplace
**Strategi:**
- List produk Shopee sebagai "Pre-order"
- Harga markup 5-10% (untuk cover effort)
- Deskripsi lengkap + foto menarik
- Link Shopee di deskripsi: "Beli langsung di Shopee lebih murah: [link]"

**Keuntungan ganda:**
- Orang beli via Marketplace → kamu dapat markup
- Orang klik link Shopee → kamu dapat komisi affiliate

### 4. Auto-Reply System
**WhatsApp/Messenger Bot:**
- Auto-reply pertanyaan umum
- Kirim link affiliate otomatis
- Follow-up setelah 24 jam (reminder)

**Template:**
```
Halo! Terima kasih sudah tertarik 😊

Produk [nama] bisa langsung dibeli di sini:
👉 [LINK AFFILIATE]

Ada diskon [X]% hari ini! Buruan sebelum kehabisan 🔥

Butuh rekomendasi produk lain? Chat aja ya!
```

## 🔥 Fase 4: Scaling & Optimization (Minggu 2+)

### 1. A/B Testing
**Test variables:**
- Caption style (formal vs casual)
- CTA (beli sekarang vs klik link)
- Waktu posting
- Jenis konten (foto vs video)
- Emoji usage

**Metric tracking:**
- CTR (Click-Through Rate)
- Conversion rate
- Komisi per post
- Engagement rate

### 2. Automation Workflow
**Daily routine (bot):**
```
06:00 - Riset produk trending (scrape Shopee)
07:00 - Generate 5 post + caption
08:00 - Post ke Facebook Page
09:00 - Post ke 10 grup (stagger 5 menit)
12:00 - Post stories (3-5 produk)
13:00 - Reply comments & DM
19:00 - Post prime time (best product)
20:00 - Analyze performance
21:00 - Schedule besok
```

### 3. Content Recycling
- Repost best-performing content (1 minggu kemudian)
- Update harga/diskon
- Refresh caption
- Tambah urgency ("Stok tinggal sedikit!")

### 4. Collaboration
- Partner dengan micro-influencer (barter produk)
- Join forces dengan affiliate lain (cross-promote)
- Buat grup WhatsApp eksklusif (early access deals)

## ⚠️ Pitfalls to Avoid

1. **Spam = Ban**
   - Jangan post terlalu sering di grup
   - Variasi konten & caption
   - Kontribusi value, bukan cuma jualan

2. **Low-Quality Content**
   - Foto blur/watermark → skip
   - Caption asal-asalan → no engagement
   - Invest di visual menarik

3. **Produk Salah**
   - Komisi rendah (<3%) → buang waktu
   - Rating jelek → refund tinggi
   - Stok sering habis → customer kecewa

4. **No Tracking**
   - Gak tau post mana yang perform
   - Gak bisa optimize
   - Setup analytics dari awal!

5. **Burnout**
   - Automate sebanyak mungkin
   - Batch content creation (1 minggu sekaligus)
   - Delegate kalau sudah profit
